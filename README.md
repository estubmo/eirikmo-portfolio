# Eirik Mo - Portfolio

A portfolio website for Senior Full-Stack Developer Eirik Mo, featuring an interactive 3D scene with scroll-driven animations.

Built with [Nuxt 3](https://nuxt.com/), [Vue 3](https://vuejs.org/), [TypeScript](https://www.typescriptlang.org/), [Three.js](https://threejs.org/), [TresJS](https://tresjs.org/), [Tailwind CSS](https://tailwindcss.com/), and [Bun](https://bun.sh/).

## ðŸ§ž Commands

All commands are run from the root of the project, from a terminal:

| Command               | Action                                           |
| :-------------------- | :----------------------------------------------- |
| `bun install`         | Install dependencies                             |
| `bun run dev`         | Start local dev server at `localhost:3000`       |
| `bun run build`       | Build production site to `.output/`              |
| `bun run generate`    | Generate static site                             |
| `bun run preview`     | Preview build locally before deploying           |
| `bun run lint`        | Check for ESLint errors                          |
| `bun run lint:fix`    | Fix ESLint errors                                |
| `bun run format`      | Check for Prettier errors                        |
| `bun run format:fix`  | Fix Prettier errors                              |
| `bun run update-deps` | Interactively update outdated dependencies       |
